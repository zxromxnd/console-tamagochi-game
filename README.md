# 🐾 Console Tamagochi Game

My first independent C++ project. This is a simple console-based Tamagochi game built using Object-Oriented Programming (OOP) principles. 

## 🎮 How to play
You are taking care of a virtual pet named **Sugar**. Your goal is to keep Sugar alive for as many days as possible by balancing its stats. 

Every time you take an action, time passes, and your pet gets a little hungrier!

**Actions available:**
* `1` - **Feed**: Decreases hunger, but makes Sugar slightly tired.
* `2` - **Sleep**: Restores energy to 100%, but Sugar wakes up hungry.
* `3` - **Play**: Fun, but burns energy and increases hunger!
* `0` - **Exit**: Close the game.

## ⚙️ Features & Tech Stack
* **Language:** Standard C++
* **Architecture:** Split files (`main.cpp`, `Pet.h`, `Pet.cpp`) for clean code and modularity.
* **Logic:** Encapsulation of pet stats, random dynamic events (random hunger increase), and status monitoring.
* **Dynamic UI:** Pet changes emotions (text emojis) based on its current stats.
